# Project Setup

## Backend

Go vers le répertoire du backend, installez les dépendances et démarrez le serveur

```bash
cd backend
npm i
npm run start
```

Pour lancer les tests backend

```cd backend
npm run test
```

Retourner en arrière pour installer les dépendances du frontend 

```bash
cd ..
cd frontend
npm i
npm run dev
```

Adresse du backend: 
http://localhost:8080

Addresse du front (devmod):
http://localhost:5173
